# Sahiti Veeravalli Portfolio

Personal portfolio site built with Vite, React, TypeScript, shadcn-ui, and Tailwind CSS.

## Getting Started

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run preview` previews the production build locally.
- `npm run lint` runs ESLint across the codebase.
- `npm run test` runs the Vitest suite.

## Project Structure

- `src/pages/Index.tsx` is the main portfolio page.
- `src/components/` contains the section components and shared UI.
- `public/` contains static assets such as the resume and favicon.
